# BOB-10 交付说明

## 范围

- 静态 Web 四国军旗游戏，默认入口为 `index.html`。
- 支持红、蓝、绿、黄四方预设阵营。
- 棋盘包含大本营、行营、铁路、公路等可玩标识。
- 支持司令、军长、师长、旅长、团长、营长、连长、排长、工兵、炸弹、地雷、军旗。
- 支持当前回合、选中棋子、合法移动提示、战斗结果、出局和胜负提示。
- 对手棋子默认暗棋展示，交战后在战报中展示必要结果。

## 规则覆盖

- 普通棋子按道路单步移动。
- 工兵可沿铁路连续移动。
- 地雷和军旗不可移动。
- 按军衔大小进行基础吃子判定。
- 炸弹交战双方同归于尽。
- 工兵可排雷。
- 军旗被夺后对应阵营出局。

## 验证命令

```sh
npm test
npm run build
npm run smoke
```

`npm run build` 会生成 `dist/app.bundle.js`，默认页面引用该 bundle，保证浏览器通过 `file://` 直接打开 `index.html` 时也能运行。
